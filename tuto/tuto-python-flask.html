tuto : crée un serveur flask python sur un raspberry pi

Etape 1 : installer les dépendances

sudo pip3 install flask --break-system-packages

ce permet de avoir le paquet python directement installer dans le système python

Etape 2 : créer le fichier app.py

crée le dossier

mkdir python-flask 
cd python-flask

puis le fichier app.py

nano app.py

puis ajoutez le code suivant :


from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template("index.html")

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)

Etape 3 : créer le dossier templates

crée le dossier templates

mkdir templates 
cd templates

puis le fichier index.html

nano index.html

puis ajoutez le code suivant :


<!doctype html>
<html lang="fr">
<head>
  <meta charset="utf-8">
  <title>Titre de la page</title>
</head>
<body>
  <h1>hello world !</h1>
</body>
</html>

Etape 4 : lancer le serveur

pour lancer le serveur, il faut faire :

python3 app.py

puis aller sur l'adresse http://votre-ip-raspberry-pi:5000/
